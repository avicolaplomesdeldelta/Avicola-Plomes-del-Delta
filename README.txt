AVÍCOLA PLOMES DEL DELTA - v6 (con protección anti-caché)
=========================================================================
Si la versión anterior parecía no actualizarse aunque subieras los
archivos, era el navegador sirviendo una copia en caché del archivo
anterior (mismo nombre). Ahora index.html pide los archivos con un
número de versión "?v=6", forzando descarga fresca.

IMPORTANTE: la PRIMERA vez tras subir esta versión, fuerza un refresco
sin caché en Chrome (mantener pulsado recargar → "Recarga forzada",
o borrar caché desde Configuración → Privacidad).

CÓMO ACTUALIZAR:
-------------------------------------
1. Descomprime, GitHub → Add file → Upload files → los 6 archivos
2. Commit changes → espera 1-2 min → recarga (fuerza sin caché esta vez)
