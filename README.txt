AVÍCOLA PLOMES DEL DELTA - App con sincronización en la nube (Firebase)
=========================================================================
Versión corregida: arregla el fallo de sincronización donde los
pollitos pendientes de anillar desaparecían al cambiar de pestaña
justo después de editar el número de eclosionados.

CÓMO ACTUALIZAR TU APP YA PUBLICADA:
-------------------------------------
1. Descomprime este zip
2. Ve a tu repositorio en GitHub
3. "Add file" → "Upload files"
4. Sube los 6 archivos: index.html, app.bundle.js, firebase-init.js,
   manifest.json, icon-192.png, icon-512.png
5. Commit changes
6. Espera 1-2 minutos y recarga tu enlace de GitHub Pages

CÓMO SABER SI ESTÁ SINCRONIZADA:
-------------------------------------
Arriba a la derecha del encabezado:
- ☁️ Sincronizado (verde) = funcionando
- ⏳ Conectando (ámbar) = espera un momento
- ⚠️ Local (rojo) = sin nube, usando copia del dispositivo
